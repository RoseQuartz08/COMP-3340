Admin Help:

Admin Login:

​

Admins can log in using username and encrypted password to get access to the sites data

​

Navigation:

​

Below you will be told everything you can do in each section of the admin area
All these options are reached through the side navigation bar after logging in

​

Dashboard:

​

The main page of the administrator section is the dashboard
Gives quick stats about the webpages data for the admin

​

Logout:

​

Dropdown for the user to log out of the admin account once they are done using the controls​


Users:

​

The admin gets a list of all the sites users where they can click to delete or edit their data


Admin can add new site users

​

Store:

​

The administrator gets a list of all the restaurants where they can click to delete or edit their data​


The admin can add a categories of food to mark where the food is from


Admin has a section to add restaurant including uploading the picture to represent them


Menu:

​

Admin gets access to the list of all dishes and can delete and edit any of them​
The administrator can add new dishes to the list assigning them to what restaurants makes them​


Orders:

​

Administrators can look over all the orders placed by the customers​


Admins can launch into editing the details of an order to update the customer of the status of their order
